# OpenClaw + Playwright Recipes for Free AI Tools

A growing collection of automation recipes — OpenClaw task configs, Playwright scripts, and minimal API quickstarts — for the **free** AI APIs, agent frameworks, and developer hosting platforms reviewed at [**toolfreebie.com**](https://toolfreebie.com).

Every recipe pairs with a long-form review of the underlying tool, so you can read the honest take and grab a working script in one place.

## Why this exists

Free-tier AI tools change quotas, rename endpoints, or move behind email walls every few weeks. This repo tracks **what still actually works for $0** in 2026 — checked, not regurgitated from marketing pages.

Each script is also a working OpenClaw / Playwright example you can copy.
